電車の中で現在のバッファのファイル名のbasenameをWikiLinkとしてコピーする拡張を書く。
このくらいならもうPC無しでも作っていけるな。

```js
/**
  現在のバッファの名前のbasenameをWikiLinkとしてコピー
*/
function wiki() {
  let buf = selected_buffer();
  let url = buf.url;

  if(buf.url) {
    let name = String(buf.name).replace(/\.md$/, "");
    copy_to_clipboard(`[${name}](https://karino2.github.io/RandomThoughts/${name})`);
  }
}
```

buf.nameがJavaのStringになっちゃってJSのにならないのはちょっとかっこわるいが。

これでWikiの記事を書くのが快適になった。

### 高齢化の話

[出生率の急低下と成長率の上昇：人口動態の変化とマクロ経済 - himaginary’s diary](https://himaginary.hatenablog.com/entry/20260703/Baby_Busts_and_Growth_Booms)

うちの親は80代になったのだが、70代があのくらいで乗り切れるならそんなに手は掛かってない。
結構良い方だと思うんだが、ある程度改善して平均がこのくらいで乗り越えられるようになれば高齢化はそんなに酷い事にならずに乗り切れる気がするな。
